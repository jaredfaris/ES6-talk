Hoisting Example

Hoisting happens because JS does two passes during interpretation. One processes variables and functions. The second actually executes the code. The first step organizes the code to make the second faster, and hoisting is part of this.

Let

The "let" keyword creates a local block scope.


    if (myVar > 5) {
        let myVarSquared = myVar * myVar;
        console.log(myVarSquared);
    }

Usable inside of if/else and loops.
